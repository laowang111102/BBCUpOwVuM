# Springboot和Vue社团管理系统 java789

## 项目概述

社团管理系统是基于Springboot和Vue的前后端分离的解决方案，旨在为高校或社团组织提供一个便捷、高效的管理平台。本系统涵盖了社团管理的多个方面，包括用户管理、社团信息、活动组织等。

## 技术栈

- 后端：Springboot、MyBatis-Plus、Maven、MySQL
- 前端：HTML、JavaScript、CSS、Vue

## 功能模块

- 系统首页
- 系统用户管理
  - 用户角色分为普通用户和管理员
- 社团类型管理
- 社团信息管理
- 社团成员管理
- 社团活动管理
- 通知信息管理
- 入团申请记录
- 费用记录管理

## 系统角色

- 用户：普通用户可以进行入团申请、查看活动通知等操作。
- 管理员：负责系统管理、社团信息维护、活动组织、成员管理等。

## 运行环境

- 支持操作系统：Windows系统、macOS
- JDK版本：1.8
- Node.js版本：14
- Maven版本：3.6及以上
- MySQL版本：5.7及以上

## 部署步骤

1. 环境准备：确保JDK、Node.js、Maven、MySQL等运行环境已正确安装。
2. 数据库配置：配置MySQL数据库，并导入初始数据。
3. 后端部署：使用Maven构建后端项目，并运行主程序。
4. 前端部署：在Vue环境下构建前端项目，并运行。

## 目录结构

```
项目根目录
├── backend                # 后端源代码目录
├── frontend               # 前端源代码目录
└── docs                   # 项目文档目录
```

## 核心亮点

- 前后端分离：提高开发效率，便于维护和扩展。
- 灵活的权限管理：区分用户角色，保障系统安全。
- 完善的功能模块：全面覆盖社团管理的各个方面，实现信息一体化管理。
